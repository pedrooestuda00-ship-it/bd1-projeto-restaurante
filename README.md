## 📂 Estrutura do Repositório
* `relatorio.pdf`: Documento contendo os Modelos ER e Relacional, Dicionário de Dados e as justificativas da Normalização.
* `trabalho_bd.sql`: Script completo (DDL e DML) contendo a criação das tabelas, as Views, a Function/Trigger e a inserção dos dados para população do banco, totalmente adaptado para **PostgreSQL**.

## 🚀 Como executar
Para testar o sistema:
1. Conecte-se ao seu servidor **PostgreSQL** (via pgAdmin, DBeaver ou terminal `psql`).
2. Crie um banco de dados (ex: `CREATE DATABASE trabalho_bd;`).
3. Abra a ferramenta de query conectada a este banco, cole o conteúdo de `trabalho_bd.sql` e execute o script integralmente.
